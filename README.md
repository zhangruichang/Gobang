Gobang
======

## Description:
A simple game of Gobang(五子棋) designed in Visual Basic 6.0.

##Requirements:
1. Visual Basic 6.0 IDE to edit the project.

##Installation:
Clone the Repository.

	git clone https://github.com/zhangruichang/Gobang
	
##Running:
Just execute the "Gobang.exe" file.
